HTML TABLE REFERENCE

Table Tags

Tag Name Description
em Emphasis This indicates Emphasis that subtly changes the meaning of a sentence
br/ Line break This causes the browser to automatically show each new paragraph or heading on a new line
blockquote Quotations This element is used for longer quotes that takes up an entire paragraph.
table Table The wrapper element for all HTML tables
width width This is used to indicate how wide the table should be
tbody table body The body should sit inside the tbody element.

Table Attributes

Tag Name Description
align Align This enumerated Attribute indicates how the table must be aligned inside the containing document
bgcolor Background color The background color of the table.
cellpadding Cell padding This attribute defines the space between the content of a cell and its border, displayed or not.
